# centipedeAssembly
A version of centipede made in assembly (MIPS)

Use these settings (MARS)
Bitmap display: 
Unit width: 8
Unit height: 8
Display width: 256
Display height: 256
Display address: $gp

MMIO keyboard simlulator

Gameplay:
Move left with J
Move right with K
Shoot with X
Restart with S
